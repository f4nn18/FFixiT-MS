# Purpose

Dokumen ini menjadi pedoman desain antarmuka FFixiT-MS agar seluruh halaman memiliki tampilan yang konsisten.

---

# Design Principles

- Clean
- Professional
- Responsive
- Consistent
- Easy to Use

---

# Framework

Gunakan:

- Bootstrap 5

Jangan menggunakan:

- Tailwind CSS
- Livewire
- Admin Template yang kompleks

---

# Layout

Gunakan struktur berikut.

Sidebar

↓

Top Navbar

↓

Content

↓

Footer

---

# Sidebar

Sidebar harus tetap konsisten pada seluruh halaman.

Menu ditampilkan sesuai Role.

---

# Dashboard

Dashboard setiap Role memiliki tampilan berbeda.

Tetapi menggunakan layout yang sama.

---

# Card

Gunakan Bootstrap Card untuk:

- Statistik
- Informasi
- Ringkasan

---

# Table

Gunakan:

Bootstrap Responsive Table

Data penting menggunakan pagination.

---

# Form

Form menggunakan:

Bootstrap Form

Label di atas input.

Gunakan placeholder seperlunya.

---

# Button

Primary

Untuk aksi utama.

Success

Untuk simpan.

Warning

Untuk edit.

Danger

Untuk hapus.

Secondary

Untuk kembali.

---

# Color

Gunakan warna Bootstrap.

Jangan menggunakan warna yang berlebihan.

---

# Icons

Gunakan:

Bootstrap Icons

---

# Responsive

Desktop menjadi prioritas utama.

Tetap mendukung Tablet dan Mobile.

---

# UI Consistency

Semua halaman harus memiliki:

- Judul Halaman
- Breadcrumb
- Action Button
- Search
- Table atau Card

---

# Empty State

Apabila data kosong.

Tampilkan pesan yang jelas.

Contoh:

"No Data Available"

---

# Loading

Gunakan loading indicator apabila proses membutuhkan waktu.

---

# Error Message

Pesan kesalahan harus:

- Jelas
- Singkat
- Mudah dipahami

---

# Goal

UI FFixiT-MS harus sederhana, profesional, cepat dipahami, dan nyaman digunakan dalam operasional toko komputer sehari-hari.

Desktop First

Minimum Resolution

1366 x 768

Optimized Resolution

1920 x 1080
